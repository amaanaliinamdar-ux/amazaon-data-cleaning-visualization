# Data Cleaning and Visualization Project

This project demonstrates data cleaning, preprocessing, and visualization techniques using Python libraries such as pandas, matplotlib, and seaborn. The dataset contains product specifications, prices, ratings, and other details.

## Features

- Data cleaning of messy columns (e.g., mixed units, NaNs)
- Conversion of units (e.g., centimeters to inches, TB to GB)
- Handling missing values with appropriate strategies
- Extraction of structured information from dictionaries and strings
- Visualization of distributions, relationships, and categorical counts using Matplotlib and Seaborn

## Getting Started

### Prerequisites

Make sure you have the following Python libraries installed:

- pandas
- matplotlib
- seaborn
- numpy
